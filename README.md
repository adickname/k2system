# Project Overview
This project was developed based on the state of https://www.ksero-k2system.pl/ (04/2025)

## Project scope
Project was developed for learning purposes 

## Technologies
- Vue
- Vue Router
- Laravel
- PrimeVue
- Tailwind CSS
- Axios
- Laravel Cashier
- Laravel Sanctum

## Authentication & Roles
- User
- Admin

## Admin Features
- Adding products to the database
- Editing products
- Deleting products
- Filtering products by name
- Notifications about request status changes

## User Features
- Adding products to the cart
- Removing products from the cart
- Editing product quantity
- Payments via Stripe (credit card or BLIK)
- Order history
- Checking whether a company is within the user's service area
